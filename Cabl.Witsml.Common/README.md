# Cabl.Witsml.Common

Utilidades compartidas entre el **visor WITSML 1.4.1** y el producto **WITSML 2.1/ETP**:

- URIs de namespace (`WitsmlNamespaces`)
- Detección de versión XML (`WitsmlXmlVersionDetector`, `WitsmlStandardKind`)

**No incluye:** parsers completos, acceso a base de datos, cliente ETP.

Referencias de proyecto: `WitsmlODViewer.csproj`, `WitsmlODViewer.WinForms`, `WitsmlEtp21/src/Witsml21.Converter`.
