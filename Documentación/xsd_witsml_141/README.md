# XSD WITSML 1.4.1 (serie `1series`)

Esquemas oficiales **WITSML v1.4** usados para:

- **Matriz literal** elemento XML (1.4.1) → columna / extractor en `Witsml141Processor`
- Validación opcional de archivos de prueba bajo el namespace `http://www.witsml.org/schemas/1series`

## Contenido típico en este repo

Incluye definiciones por objeto, por ejemplo:

- `obj_well.xsd`, `obj_wellbore.xsd`, `obj_trajectory.xsd`, `obj_mudLog.xsd`, `obj_log.xsd`, …
- Tipos compartidos (`cs_*`, `grp_*`) según el paquete Energistics copiado.
